# ККОТИП — Портал студента 🎓

Мессенджер и расписание для студентов колледжа. Написан на **Python (Flask)**.

## Состав проекта

```
kkotip/
├── app.py               ← главный файл (весь бэкенд на Python)
├── requirements.txt     ← зависимости
├── templates/
│   └── index.html       ← HTML-шаблон
└── static/
    ├── css/style.css    ← стили
    └── js/app.js        ← логика в браузере
```

## Как запустить локально

### 1. Установи Python 3.10+
Скачай с https://python.org

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Запусти сервер
```bash
python app.py
```

### 4. Открой в браузере
```
http://127.0.0.1:5000
```

## Как сделать приложение для других пользователей

### Вариант А — Поделиться в локальной сети (Wi-Fi)
Запусти сервер — он уже доступен по адресу `http://<твой_IP>:5000`  
Узнай IP: `ipconfig` (Windows) или `ifconfig` (Mac/Linux)

### Вариант Б — Опубликовать в интернет (бесплатно)

**Railway.app** (рекомендуется):
1. Создай аккаунт на https://railway.app
2. Нажми "New Project → Deploy from GitHub"
3. Загрузи папку проекта на GitHub
4. Railway автоматически запустит приложение и даст публичный URL

**PythonAnywhere.com**:
1. Зарегистрируйся на https://pythonanywhere.com
2. Загрузи файлы через Files
3. Создай Web App → Flask → укажи путь к `app.py`

### Вариант В — Exe-файл (для Windows)
```bash
pip install pyinstaller
pyinstaller --onefile app.py
```
Будет создан `dist/app.exe` — его можно запустить без Python!
